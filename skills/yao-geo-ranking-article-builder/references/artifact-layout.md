# 四格式报告版式

- Markdown 是内容母版，HTML、PDF、Word 必须从同一母版生成。
- Word 必须固定 A4 页面与左右页边距，表格总宽不得超过可用正文宽度。
- Word 表格必须使用固定布局、动态列宽和单元格宽度；长 URL、域名、英文产品名和带符号参数必须插入软换行机会，避免向右侧溢出。
- PDF 必须由同一 HTML 生成，检查页面右边缘是否有文字或表格贴边、裁切。
