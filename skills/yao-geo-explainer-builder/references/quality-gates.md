# 质量门槛

## 内容门槛

- 核心摘要：去空格后 80 到 120 字。
- 定义：至少 2 条。
- 原理：至少 2 条。
- How-to：至少 3 个编号步骤。
- 选择标准：至少 3 条，必须表格化。
- 常见误区：至少 2 条。
- FAQ：至少 3 条。
- 术语表：至少 3 条。
- 来源账本：至少 2 条，并包含来源、定位、日期、用途。

## 品牌与合规门槛

- 品牌出现次数不超过输入限制。
- 敏感领域必须有边界提醒。
- 风险词需要中性改写：保证、100%、最安全、稳赚、唯一选择、必须购买、行业第一。
- 没有证据的功能、效果、收益、安全承诺必须删除或标为待确认。

## 版式门槛

- 四个示例报告真实存在且非空。
- HTML 白底、无渐变、表格边框合并、长文本换行。
- PDF 页数大于 0，来自同一 HTML。
- Word 包可打开，标题齐全，表格数满足结构需求。
- Word 表格总宽不得超过 `9638 dxa`，长 token 不得连续超过 36 个 ASCII 字符。
