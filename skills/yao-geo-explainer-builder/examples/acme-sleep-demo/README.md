# Acme Sleep Demo

这是 `yao-geo-explainer-builder` 的合成示例，用于验证 GEO 科普文章、How-to 清单、FAQ、术语表、品牌自然植入建议和 Word/PDF/HTML/Markdown 四格式输出。

生成命令：

```bash
python3 skills/yao-geo-explainer-builder/scripts/render_yao_geo_explainer_builder.py \
  --input skills/yao-geo-explainer-builder/examples/acme-sleep-demo/report_input.json \
  --output-dir skills/yao-geo-explainer-builder/examples/acme-sleep-demo/rendered
```
