# 研究支撑框架

本 skill 的理论基础是把旧内容改造成更容易被生成式引擎检索、理解、引用和核验的知识资产。

| 研究方向 | 对本 skill 的约束 |
|---|---|
| GEO | 目标是生成式引擎可见性、可引用性和领域差异，不承诺传统 SEO 排名。 |
| RAG | 内容必须有可检索标题、定义、事实段、表格和来源锚点。 |
| 带引用生成 | 答案和证据之间要能断言级匹配。 |
| Lost in the Middle | 关键答案、定义、边界和证据摘要必须前置。 |
| FActScore | 复杂结论要拆成原子事实卡逐项核验。 |
| Self-RAG | 生成后必须自 review，检查事实支持、引用质量和输出排版。 |
