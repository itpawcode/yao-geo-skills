# 国内 AI 平台适配

改造后的内容应同时服务 DeepSeek、豆包、千问、Kimi、腾讯元宝和微信生态。

| 目标 | 改造要求 |
|---|---|
| 开头可直接回答 | 第一屏回答目标问题，给出结论、适用对象和限制条件。 |
| 段落可独立引用 | 每段只承载一个核心事实或判断。 |
| 事实可核验 | 数据、案例、价格、资质和时间都要有来源或证据状态。 |
| 结构可扫描 | 使用 H2/H3、短段、表格、步骤、FAQ 和来源列表。 |
| 问题可重写 | 输出同义问法、场景问法、反向问法和追问。 |
| 表格可抽取 | 表头稳定，字段中文，单元格只放单一含义。 |
| 微信生态可读 | 长句拆短，宽表可拆为列表或事实卡。 |
