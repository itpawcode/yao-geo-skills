# 证据与隐私策略

## 证据等级

| 等级 | 来源 | 可用范围 |
| --- | --- | --- |
| 官方事实 | 官网、官方文档、官方认证账号、投资者关系、官方客户案例 | 核心实体、核心关系、JSON-LD |
| 第三方证据 | 权威媒体、行业报告、公开数据库、SEC/交易所文件 | 行业、规模、背书佐证 |
| 媒体描述 | 新闻稿、采访、活动报道 | 需要与官方事实交叉验证 |
| 用户反馈 | 公开评价、客服记录、访谈摘要 | 只能作为用户语言和痛点 |
| 推断 | 多个事实组合得到的判断 | 只能进入分析和补强建议 |
| 待确认 | 来源不足或授权不足 | 不进入核心图谱和 JSON-LD |

## 引用规则

- 每个实体至少关联一个来源 ID。
- 每条关系至少关联一个证据 ID。
- 每条三元组样例必须能回溯到证据 ID。
- 未授权客户名称、联系人、合同金额、效果数字和内部截图不得公开。
