# China Platform Sampling

| 平台 | 重点记录 | 风险 |
|---|---|---|
| DeepSeek | 结论稳定性、证据链、推理摘要、是否联网、对比回答的一致性 | 多轮上下文影响较大，不能混用单轮和多轮样本 |
| 豆包 | 口语问答、图文输出、短答案推荐、图片或卡片是否承载品牌信息 | 多模态输出需要截图和文本双记录 |
| 千问 | 引用源、追问路径、阿里生态或电商信源、结构化结果 | 追问会改变引用和排序，必须记录轮次 |
| Kimi | 深度研究、长文引用、文档站、报告型答案、网页摘要 | 长文答案引用多，需按句子或段落判断支持度 |
| 元宝 | 微信生态来源、公众号文章、腾讯生态页面、视频号线索 | 生态内来源可能难以稳定公开访问，要记录可复核路径 |

同一轮样本固定地区、设备、账号状态和联网状态；联网和非联网分开统计；追问样本使用 `turn_index` 单独建模。
