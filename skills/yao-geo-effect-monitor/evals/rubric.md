# Evaluation Rubric

| 维度 | 通过标准 |
|---|---|
| 路由准确 | 监测、引用、纠偏、月报、告警、归因请求会触发；单纯内容生产或 CRM 追踪不触发。 |
| 平台适配 | DeepSeek、豆包、千问、Kimi、元宝均有独立采样字段和解释。 |
| 指标完整 | 包含品牌出现率、候选率、推荐率、排序、竞品、负面、描述准确、引用召回、引用准确和稳定性。 |
| 归因谨慎 | 使用基线、观察窗口、处理/对照 Prompt、竞品对照和混杂因素，不做过度因果判断。 |
| 纠偏闭环 | 问题能映射到知识库、内容、页面、外部发布或销售口径，并有复采验收。 |
| 交付质量 | 四格式文件真实存在，白底、表格有边框、不溢出，结构一致。 |
