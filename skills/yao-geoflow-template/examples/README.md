# yao-geoflow-template examples

This skill maps a reference site onto a live GEOFlow frontend contract, so full examples usually depend on a real GEOFlow workspace and a reference URL.

Use this directory for sanitized preview-first theme package examples. A publishable example should include:

- a short brief describing the reference site and target GEOFlow modules
- `tokens.json` for visual tokens
- `mapping.json` for reference-to-GEOFlow module mapping
- `manifest.json` for the generated theme package
- preview screenshots or static preview files when they do not expose private project data

Do not commit live customer themes, private URLs, admin paths, credentials, or copied third-party assets without permission.
