# yao-geo-page-blueprint 评分标准

| 维度 | 满分条件 |
| --- | --- |
| 页面类型判断 | 准确选择页面类型，并说明信息架构取舍。 |
| AI 可抽取性 | 包含键值对、表格、步骤、原子事实、实体关系、问答和上下文无关摘要。 |
| 研究依据 | 将 GEO、长上下文、RAG、结构化特征和 Schema 规则转成可执行页面原则。 |
| 证据台账 | 每个关键结论有来源、核验日期、用途和对应模块。 |
| Schema 与 CTA 边界 | Schema 只取正文可见事实，CTA 不干扰核心答案、证据区和比较表。 |
| 四格式排版 | Word、PDF、HTML、Markdown 白底、对齐稳定、表格可读、无溢出；Word 有固定表格布局、显式列宽、`wordWrap` 和长串软换行策略。 |
