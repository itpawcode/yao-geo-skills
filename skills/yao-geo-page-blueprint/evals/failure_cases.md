# 常见失败案例

- 首屏只有营销口号，没有直接答案和关键事实。
- 页面模块看起来漂亮，但 HTML 语义不清。
- FAQ 全部堆在页面底部，问题少且像广告文案。
- Schema 写了价格、评分、资质或案例，但正文没有这些内容。
- 把 FAQPage 写成 Google 富结果或流量提升承诺，而不是正文可见问答的语义候选。
- 证据区只有链接列表，没有说明来源用途、核验日期和对应页面模块。
- 强 CTA 插在核心答案或证据区中间。
- 只交付 Markdown，没有生成 Word、PDF、HTML。
- Word 表格依赖自动列宽，URL、英文产品名、Schema 字段或代码片段向右溢出。
- PDF 由未校验的 HTML 直接打印，表格右侧被截断或页面底色不是白底。
