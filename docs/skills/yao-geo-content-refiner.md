# yao-geo-content-refiner

`yao-geo-content-refiner` 用于把已有 SEO 文章、公众号文章、官网文章、白皮书和产品页文案改造成结构化、可信、可引用、可抽取的 GEO 内容。

## 核心能力

- 原文 GEO 8 维评分。
- 原子事实卡与证据账本。
- GEO 改造版文章、差异报告、FAQ 与同义问法。
- 页面发布版 HTML 建议。
- Word、PDF、HTML、Markdown 四格式输出。
- Word/PDF 防右溢出排版：宽表自动转纵向事实卡，表格固定宽度。
