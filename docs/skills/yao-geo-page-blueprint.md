# yao-geo-page-blueprint

`yao-geo-page-blueprint` 是页面技术类 GEO skill，用于基于内容和品牌目标生成 GEO 友好的页面结构、模块、信息架构、Schema、HTML 和 CMS 字段方案，并默认支持 Word、PDF、HTML、Markdown 四件套交付。

## 质量门

- 首屏必须有直接答案、关键事实和摘要。
- 必须包含研究依据与页面设计原则。
- 必须包含证据区与来源台账。
- Schema 必须与页面正文事实一致。
- FAQPage 必须对应正文可见的真实问答，不能用于广告。
- CTA 不能干扰核心答案、证据区和比较表。
- 四格式报告必须真实存在、白底、对齐稳定、表格可读、无溢出。
- Word 必须使用固定表格布局、显式列宽、单元格边距和 `wordWrap`；存在长英文、URL、Schema 字段或代码片段时必须软换行。
